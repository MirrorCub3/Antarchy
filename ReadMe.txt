ASSETS FOR CHARACTERS- Original characters drawn by Joyce and colored in by Lyndsey. 
ASSETS FOR BACKGROUND- Some backgrounds are drawn by Samantha. Others from Pixabay but blurry.

KEYBOARD INPUT- WASD/Arrow Keys TO MOVE IN BEACH GAME; Spacebar to forward text
MENU- There is a menu system for the game with options like start, continue, settings, and quit. 
SETTINGS MENU- There is a separate menu for changing the volume and speed of reading text. 

NARRATIVE- The game is a dating sim so it is narrative-based.

PROGRESS- The game also has a progress saving system that contains the player's input. For example, the songwriting game saves the player's input and displays their answers in the end. 
	- The player options of volume and text speed are saved too.
        - Game progress such as the current day and heart value for both characters is saved daily. 

HEART BARS- The main characters in the story have heart bars next to their icons. This bar goes up or down depending on the player's choices. 

SOUND- The game has continuous background music that sometimes changes based on the scene.
